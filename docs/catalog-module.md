# Module catalogue

Centraliser les articles, services, materiaux et extras qui peuvent etre ajoutes aux devis et factures.

## Champs principaux

- Item
- Categorie
- Unite
- Prix CAD
- Taxable
- Notes

## Unites importantes

- Heure
- Unite
- Pied carre
- Pied lineaire
- Boite

- Forfait

## Categories possibles

- Materiaux
- Services
- Outils
- Location
- Extra
- Main-d'oeuvre

## Fonctions attendues

- Ajouter item
- Modifier item
- Dupliquer item
- Export CSV
- Rechercher item
- Inserer dans facture
- Inserer dans devis
