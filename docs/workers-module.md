# Module employes

## Objectif

Gerer les travailleurs internes de l'entreprise.

## Champs principaux

- Nom legal
- Telephone
- Email
- Adresse
- Date embauche
- Poste
- Type paie
- Taux CAD
- Contact urgence
- Telephone urgence
- Lien urgence
- Deductions
- Avantages

## Types de paie

- Heure
- Pied carre
- Job
- Forfait

## Regles

- Un employe doit avoir un role.
- Un employe peut etre assigne a un chantier.
- Un employe doit voir seulement son propre espace.
- Les donnees privees doivent etre limitees aux admins autorises.
