# Module themes

## Objectif

Permettre a chaque compagnie ou utilisateur de choisir un visuel adapte a ses preferences.

## Themes prevus

- Carbon Gold
- Storm Steel
- Arctic Ledger
- Inferno Pro
- Ocean Glass
- Forest Work
- Neon Grid
- Classic Light

## Regles

- Les themes doivent utiliser des variables CSS.
- Le changement de theme ne doit pas casser les formulaires.
- Le theme doit rester lisible sur telephone, tablette et chantier.
- Le theme choisi doit etre sauvegarde par utilisateur plus tard.

## Prochaine etape

Ajouter les themes dans les reglages et les sauvegarder dans la base.
