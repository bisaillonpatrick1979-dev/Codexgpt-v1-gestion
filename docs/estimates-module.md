# Module devis

## Objectif

Creer des estimations de travaux avant contrat ou facture.

## Champs principaux

- Numero
- Client
- Chantier
- Description
- Montant CAD
- Statut
- Notes

## Statuts possibles

- Brouillon
- Envoye
- Accepte
- Refuse
- Expire

## Fonctions attendues

- Ajouter devis
- Modifier devis
- Dupliquer devis
- Rechercher devis
- Export CSV
- Imprimer ou enregistrer PDF
- Convertir en contrat
- Convertir en facture

Ajouter des lignes detaillees de devis avec catalogue, taxes, remise et signature.
