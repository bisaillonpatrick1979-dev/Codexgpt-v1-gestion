# Module facturation

## Objectif V1

Faire des factures utilisables localement avant Supabase.

## Champs minimum

- Numero
- Client
- Chantier
- Date
- Montant CAD
- Statut
- Notes
- Signature

## Statuts

- Brouillon
- Envoyee
- Partiellement payee
- Payee
- En retard
- Annulee

## Fonctions actuelles

- Ajouter
- Modifier
- Dupliquer
- Supprimer
- Rechercher
- Export JSON
- Export CSV
- Imprimer ou enregistrer en PDF via le navigateur

## Prochaines fonctions

- Lignes de facture detaillees
- Taxes configurables
- Remise
- Avance ou depot
- Signature tactile
- Logo
- Filigrane
- Photos chantier
- Envoi email ou texto
