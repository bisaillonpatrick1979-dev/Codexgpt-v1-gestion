# Module clients

## Objectif

Gerer les clients de l'entreprise.

## Types de clients

- Personne
- Compagnie

## Champs principaux

- Type client
- Nom personne
- Nom compagnie
- Contact principal
- Telephone
- Email
- Adresse
- Notes

## Regles

- Si le client est une compagnie, le contact principal est important.
- Un client peut etre lie a plusieurs chantiers.
- Un client peut avoir plusieurs devis, contrats et factures.

Ajouter une fiche client detaillee avec historique.
