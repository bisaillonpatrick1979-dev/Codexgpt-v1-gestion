# Batch de developpement

Objectif: travailler sur une branche separee pour eviter un redeploiement Vercel a chaque petit commit.

## Ajouts du batch

- Export CSV par module.
- Duplication d'entrees.
- Statuts rapides pour factures, devis, contrats, chantiers, recompenses et factures employe.
- Styles ameliores pour actions, mini boutons et impression.
- Preparation pour ajout futur de photos URL et pieces jointes.

## Workflow recommande

1. Creer une branche de travail.
2. Faire plusieurs commits sur cette branche.
3. Tester visuellement.
4. Merger dans main seulement quand le lot est pret.
5. Laisser Vercel redeployer une seule fois.

- Creation de lignes de facture detaillees.
- Calendrier cliquable avec details de jour.
