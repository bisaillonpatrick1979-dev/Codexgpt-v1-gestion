# Module IA chantier

## Objectif

Ajouter un assistant de chantier pour noter les questions, hypotheses, pistes de solution et suivis.

## Champs principaux

- Question
- Chantier
- Categorie
- Reponse locale
- Statut
- Notes

## Categories possibles

- Installation
- Materiaux
- Securite
- Organisation
- Probleme client
- Meteo
- Qualite
- Autre

## Fonctions V1 locale

- Ajouter une question
- Modifier une question
- Changer le statut
- Rechercher dans les questions
- Export CSV

## Production future

- Brancher une API IA via backend ou fonction serveur.
- Ne jamais mettre de cle secrete dans le frontend.
- Garder historique par chantier si l'utilisateur l'accepte.
