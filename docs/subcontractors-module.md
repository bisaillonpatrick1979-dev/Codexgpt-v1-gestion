# Module sous-traitants

## Objectif

Gerer les compagnies ou travailleurs externes qui facturent l'entreprise.

## Champs principaux

- Nom compagnie
- Contact
- Telephone
- Email
- Adresse
- Mode paie
- Taux CAD
- Fournit outils
- Peut engager aide
- Contrat signe
- Notes

## Modes de paie

- Heure
- Pied carre
- Job
- Forfait

## Regles

- Le sous-traitant doit avoir un contrat ou une entente.
- Le sous-traitant peut envoyer ses propres factures.
- Le sous-traitant ne doit pas voir les donnees internes de l'entreprise.
- Le sous-traitant doit voir seulement ses chantiers et ses factures.

## Prochaine etape

Ajouter portail sous-traitant avec factures envoyees a l'entreprise.
